# Support

NotDone is currently pre-alpha.

- Use GitHub Discussions for usage questions and design feedback when available.
- Use GitHub Issues for reproducible bugs and feature requests.
- Use the private process in [SECURITY.md](SECURITY.md) for vulnerabilities.

When reporting a runtime integration problem, include the NotDone version, runtime name and version, operating system, `notdone doctor` output, and the smallest reproducible task contract. Remove secrets before posting.
