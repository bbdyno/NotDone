# Documentation localization

## Locales

| Locale | File |
| --- | --- |
| English | `README.md` |
| Korean | `README_KO.md` |
| Japanese | `README_JA.md` |
| Simplified Chinese | `README_ZH-CN.md` |
| Traditional Chinese | `README_ZH-TW.md` |

`README.md` is the English source document. Every locale must remain a complete guide rather than a short redirect.

## Synchronization

- Keep the `docs-revision` marker equal across all README files.
- Preserve the five-language navigation block.
- Keep install commands, command names, runtime names, statuses, and license identifiers exact.
- Update every locale in the same pull request when user-visible behavior changes.
- Run `pnpm docs:check`.

Translations should use native developer terminology. Do not translate identifiers, commands, schema fields, result states, or file paths.
